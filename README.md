# postgres_read_user
To grant read permission on all databases of Aurora DB to read_user
### Clone it
### Give execute permission to the script
> chmod +x postgres.sh
### Run the script
> ./postgres.sh
### Give
 - Host/Endpoint name
 - Admin User name
 - Admin Password
 - User to grant read permission <br />
If all grant are success then you will get 0 as exit message 
Good luck :)
